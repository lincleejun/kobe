```bash
node --experimental-strip-types --no-warnings scripts/perf-stress.ts > docs/perf/baseline.md
```

Why node, not bun: node-pty's `onData` callback never fires under
Bun (same incompat that forces vitest behavior tests to run under
node). Requires node >= 22.6 for `--experimental-strip-types`.

Numbers vary with host load, terminal emulator, and CI runners.
Hosted runners (GitHub Actions, etc.) produce different numbers and
are NOT comparable to local runs. Re-baseline on the same machine when
looking for regressions.
